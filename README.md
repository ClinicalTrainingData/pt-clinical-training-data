# PT Clinical Training Data

**De-identified physical therapy initial evaluation records for AI and machine learning model training.**

---

## Overview

This repository describes a commercial dataset of de-identified physical therapy (PT) initial evaluation (IE) records authored by licensed Doctors of Physical Therapy (DPTs) in active outpatient clinical practice.

The dataset is designed for AI/ML teams building healthcare language models, clinical NLP systems, ambient documentation tools, medical coding AI, and musculoskeletal health applications.

---

## Why PT Clinical Data?

Physical therapy documentation contains a distinct clinical language that is significantly underrepresented in most healthcare AI training corpora:

- Musculoskeletal assessment terminology
- Functional outcome and deficit language
- Orthopedic special test documentation
- DPT clinical reasoning and impression narratives
- Structured plan of care with frequency, goals, and prognosis

This vocabulary does not appear meaningfully in primary care notes, hospital discharge summaries, or general EHR datasets. If your model needs to understand, generate, or process rehab or MSK clinical text — this dataset fills that gap.

---

## Dataset Contents

Each record is a complete PT initial evaluation and includes:

| Section | Description |
|---|---|
| Chief Complaint & HPI | Patient-reported onset, mechanism, pain descriptors, prior treatment history, functional limitations |
| Objective Physical Examination | Range of motion, strength testing, special orthopedic tests, posture and gait assessment, palpation findings |
| Assessment & Clinical Impression | DPT-authored clinical reasoning, working diagnosis, functional deficit summary |
| Plan of Care | Treatment frequency, goals, intervention approach, prognosis |
